# 📦 DiarioCash - Gestión de Finanzas para Repartidores

**DiarioCash** es una aplicación móvil desarrollada con **Flutter** diseñada específicamente para repartidores y trabajadores independientes. Permite llevar un control estricto de los ingresos y gastos diarios de forma rápida y sencilla.

## ✨ Características principales
* 💰 **Registro de movimientos:** Añade rápidamente ganancias y gastos.
* 📅 **Agrupación inteligente:** Los movimientos se agrupan automáticamente por fecha (Hoy, Ayer, etc.).
* 📊 **Resumen de saldo:** Visualización clara de ingresos totales, gastos y balance neto.
* 🚀 **Interfaz intuitiva:** Diseñada para ser usada en movimiento, con una navegación fluida.
* 🎨 **Personalización:** Logo propio y tema oscuro optimizado.

## 🛠️ Tecnologías utilizadas
* [Flutter](https://flutter.dev/) - Framework de UI.
* [Dart](https://dart.dev/) - Lenguaje de programación.
* [Intl](https://pub.dev/packages/intl) - Para el formato de fechas y moneda.
