📦 DiarioCash
Aplicación móvil creada con Flutter para ayudar a repartidores a gestionar sus ingresos y gastos diarios.

📅 Agrupación de gastos por fecha.

📊 Resumen de saldos.

🎨 Personalización completa con logo propio.
